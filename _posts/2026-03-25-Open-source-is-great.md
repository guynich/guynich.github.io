Today I found ["Uptime Kuma<sup>1</sup>" on GitHub](https://github.com/louislam/uptime-kuma) thanks to Claude.

It's an open-source monitoring tool for checking the uptime of networked services.  It can generate alerts when something is down.  In the past I have written custom scripts which can all now be replaced/deprecated by a much better tool.

I mention this because it reminded me, once again, of the greatness of open-source software.  Open-source projects help each other make better things.  Fantastic!

<sup>1</sup> Kuma — means "bear" in Japanese.
